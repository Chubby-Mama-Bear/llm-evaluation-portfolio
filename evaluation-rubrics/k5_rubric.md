# K–5 Content Evaluation Rubric

These rubrics are illustrative of my evaluation approach for K–5 content.

| Criteria             | Description                                             |
|----------------------|---------------------------------------------------------|
| Accuracy             | Facts are correct, no misleading statements            |
| Grade-Appropriateness| Vocabulary + concepts match ages 6–11                 |
| Clarity              | Explanations are simple, direct, and child-friendly    |
| Engagement           | Encourages interest and participation                  |
| Safety               | No sensitive or inappropriate content                  |

Score each from 1–5 and record trends over time.
