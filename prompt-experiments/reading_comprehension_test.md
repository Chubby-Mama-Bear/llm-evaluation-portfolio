# Reading Comprehension Evaluation (Grades 1–4)

## Objective
Assess LLM's ability to generate reading comprehension questions aligned with early-elementary literacy standards.

## Prompt Tested
“Create a short reading passage for a 2nd grader about a child planting a seed. 
Include 3 comprehension questions.”

## Evaluation Criteria
- ✔ Vocabulary appropriate for age 6–9  
- ✔ Clear sentence structure  
- ✔ Meaningful comprehension questions  
- ✔ No advanced or abstract concepts  

## Observations
- Passage length appropriate  
- Vocabulary aligned to grade level  
- Questions require recall and basic inference  

## Recommendation
Add rubric scoring for sentence complexity and word frequency.
