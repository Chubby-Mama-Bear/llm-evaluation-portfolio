# Mandarin Vocabulary Evaluation — Early Learners (Ages 6–9)

## Objective
Test LLM accuracy when generating beginner Mandarin vocabulary and repetition-based practice exercises.

## Prompt Tested
“Create a practice activity for children learning Mandarin (ages 6–9) 
focusing on basic words: apple (苹果), dog (狗), water (水). Include repetition exercises.”

## Evaluation Criteria
- ✔ Correct characters  
- ✔ Correct pinyin  
- ✔ Age-appropriate repetition style  
- ✔ Simple and clear practice instructions  

## Observations
- Vocabulary correct  
- Pinyin accurate  
- Repetition supports memory retention  
- Good alignment with 'learn + practice' teaching method  

## Improvements
- Add spaced repetition steps  
- Add a trace-the-character element for handwriting practice
