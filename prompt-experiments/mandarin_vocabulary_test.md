# Mandarin Vocabulary Prompt Experiment (K–5)

**Experiment metadata**  
- Date: Example  
- Context: Beginner Mandarin vocabulary and repetition-based practice exercises  
- Goal: Evaluate accuracy and clarity of exercises for an ages 6–9 audience  

## Mandarin Vocabulary Evaluation — Early Learners (Ages 6–9)

## Objective

Test LLM accuracy when generating beginner Mandarin vocabulary and repetition-based practice exercises.

## Prompt Tested

“Create a practice activity for children learning Mandarin (ages 6–9) focusing on basic words:  
apple (苹果), dog (狗), water (水). Include repetition exercises.”

## Evaluation Criteria

- ✔ Correct characters  
- ✔ Correct pinyin  
- ✔ Age-appropriate repetition style  
- ✔ Simple and clear practice instructions  

## Observations

- Vocabulary correct  
- Pinyin accurate  
- Repetition supports memory retention  
- Good alignment with **learn + practice + repetition** teaching method  

## Improvements

- Add spaced repetition steps  
- Add a trace-the-character element for handwriting practice
